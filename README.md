# Next Elite Academy - React App

A modern educational platform website built with React and Vite.

## Features

- Course catalog and details
- Blog system with social media feed
- Contact and enrollment forms
- Admin CMS panel
- Responsive design with Tailwind CSS

## Tech Stack

- React 19.2.0
- Vite 6.2.0
- Firebase (Analytics)
- Tailwind CSS
- Lucide React (Icons)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
react-app/
├── components/      # Reusable UI components
├── views/          # Page components
├── firebase/       # Firebase configuration
├── App.jsx         # Main application component
├── index.jsx       # Entry point
└── constants.js    # Initial data
```

## Firebase Configuration

Firebase is configured for the `nextelitefnweb` project. See `firebase/config.js` for details.
